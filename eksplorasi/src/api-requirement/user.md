# Untuk halaman manajemen pengguna sesuaikan dengan schema user

- post untuk create user
- get list user
- get user by id
- put untuk update user
- delete untuk delete user

## Untuk authentication

- post untuk login (dengan email dan password)
- post untuk logout (dengan token)
- post untuk refresh token (dengan refresh token)
- get untuk mendapatkan data user yang sedang login (dengan token)
