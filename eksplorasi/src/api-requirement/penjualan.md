# Kebutuhan halaman penjualan (Analitik Penjualan)

## Untuk masing-masing card:

- omzet 30 hari: berikan angka omzet dan persentase kenaikan / penurunan dari 30 hari sebelumnya, gunakan 2 bulan. 30 hari ke belakang (bulan ini) untuk ditampilkan, dan bulan lalu untuk perbandingan
- Total transaksi: seluruh total transaksi (total order) dan berikan total item terjual seluruhnya
- Rata-rata pembelian: Nilai belanja per-transaksi

## Tren pendapatan 30 hari terakhir

data yang akan ditampilkan:

- omset untuk grafik y
- hari untuk grafik x
- untuk pendapatan tertinggi nanti akan di highlight dengan garis vertical sebagai puncak penjualan (mungkin ini biarkan frontend handle saja)

## Produk terlaris - dalam bentuk diagram batang

- nama produk untuk grafik y
- total penjualan untuk grafik x

## Traffic Hour juga dibutuhkan di sini

## Kontribusi kategori - dalam bentuk donnut chart

- Nama kategori
- Persentase kontribusi penjualan

## Riwayat transaksi - dalam bentuk tabel

- id pesanan
- waktu
- produk (format tiap produk: "<qty>x <nama_produk>", pisahkan dengan koma)
- total harga transaksi
